# Flows

## User flow
1. User signs up
2. User logs in
3. User browses services
4. User selects a service
5. User checks out
6. User pays
7. User sees payment result
8. User views order in dashboard

## Admin flow
1. Admin logs in
2. Admin views orders
3. Admin updates order status
4. Admin adds notes
5. Admin manages services
6. Admin checks payments
